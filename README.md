# AI-Powered Recruitment Pipeline

Built with n8n, Google Gemini, Gmail, Google Sheets, and JavaScript.

## What it does
- Screens CVs using Google Gemini 2.5 Pro (scored 0-100)
- Sends daily HR digest email with Approve/Reject buttons
- Token-secured one-time approval links
- Automatically emails candidates based on HR decision
- Updates Google Sheets CRM with candidate status

## Workflows
- `recruitment-pipeline.json` — main CV screening workflow
- `daily-hr-digest.json` — daily digest + approve/reject handler
